# assets/faces/

Baked face point clouds (.npz) for the face-cloud base layer, created by
double-clicking "Capture Face.sh". Each file is a small set of 3D points +
colours sampled from one webcam capture (MediaPipe Face Mesh model via
onnxruntime, run offline in its own venv).

In the VJ app: press the ` (backtick) key to show the face cloud, , / . to
cycle faces, and the arrow keys to turn/tip the head.

These are operator media and are gitignored (like assets/clips/).
