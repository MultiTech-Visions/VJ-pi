# assets/images/

Static image files (.png, .jpg, .jpeg) used to texture the
**imagedonut** generator. Drop your image in here and it'll be
applied to the donut's surface on the next launch.

The app picks the first image alphabetically. If you want a
specific one, name it starting with `0_` or similar so it sorts
first.

If the folder is empty, the app auto-generates a colourful
6-colour checker pattern at `_default_checker.png` so the donut
generator works out of the box. You can delete that file or
ignore it — your images take priority.
